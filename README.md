# Frontend of a Restaurant Menu Page

## 1. Project Title
Frontend of a Restaurant Menu Page

## 2. Project Objective
The objective of this project is to design and build a responsive, attractive, and user-friendly restaurant menu website using HTML, CSS, and JavaScript. The website displays food items category-wise and allows users to filter items using JavaScript.

## 3. Project Description
This project is a frontend-only restaurant menu website.

- Menu items are displayed under categories such as Starters, Main Course, Desserts, and Beverages.
- JavaScript is used to filter menu items by category.
- The website is responsive and works on all screen sizes.

## 4. Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- CSS Flexbox
